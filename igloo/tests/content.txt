========================================================
Content types in Igloo
========================================================

Basic Usage
------------------------

Any content you like which is an Axiom Item can be stored in the database, 
however sometimes you want to expose some of these items to the Igloo admin.

By creating an IContentType adapter, you can do that.

So lets start by creating a store.
 
    >>> from axiom.store import Store
	>>> from axiom.item import Item
    >>> store = Store()

Now, let's create a class to represent an article:

	>>> from axiom.attributes import text, reference
	>>> from axiom import dependency
	>>> from igloo.content import ContentType
	>>> class Article(Item):
	... 	"""A News Article"""
	... 	title = text()
	... 	author = text()
	... 	text = text()
	... 	contentType = reference()
	... 	def __init__(self, site, **kw):
	... 		Item.__init__(self, **kw)
	... 		self.contentType = self.store.findOrCreate(ContentType, site=site, path=u"news", name=u"News Article")
	... 		dependency.installOn(self.contentType, self)

and so we can instantiate an article:

	>>> from igloo.content import Site
	>>> site = Site(store=store, title=u"Tech News")
	>>> articleNevow = Article(store=store, site=site, title=u"Introducing Nevow, one kickass web framework.")
	>>> list(site.getContentTypes())
	[News Article]
